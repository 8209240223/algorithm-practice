# 算法练习

Visual Studio C++ 小练习：蓝桥相关题目源码（如 `2015.9.cpp`、`2016-a-10.cpp`）。

## 构建

用 Visual Studio 打开 `practice1.sln`，选择配置后生成 `Project1`。

## 推送到 GitHub

若本机未安装 GitHub CLI，请在网页新建仓库（仓库名需为英文/数字/连字符，例如 `algorithm-practice`），然后：

```bash
git remote add origin https://github.com/<你的用户名>/algorithm-practice.git
git branch -M main
git push -u origin main
```
